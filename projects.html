Frank Chen

Hackathon Projects


Myodrone

Use Myo armband to control AR Parrot drone with 8 degrees of movement. Photos/videos taken by the drone are accessed by the Intel Edison & forwarded to a web app, where they can be easily viewed across multiple platforms.

Top 30 out of 300+ teams at PennApps Spring 2015.

MyoDrone on Github

RoboHome

Internet of Things (IoT) themed hardware project: model home with components such as lighting, door, TV, and shower. Each of these components were connected to a microcontroller + bluetooth module; they were controlled using a variety of mobile devices: smartphone, pebble watch, myo armband, and laptop.

First Place out of 200+ competitors at IDEA Hacks 2016

RoboHome on Github

Alexa MD

Hardware IoT project with three components: Amazon Echo prompt the user with various versions of exposure therapy. The user will then engage in VR treatment via Oculus Rift by re-enacting situations that may usually cause them anxiety/distress.

Won Best Use of Amazon Echo and Best Health Wearable Prize at TreeHacks 2016

Alexa MD on Github

MyoMessenger

Hardware driving safety project using Myo armband to send common text messages to any person on a predetermined emergency list using using hand gestures. The Muse headband’s accelerometer detect when a driver’s head starts bobbing in exhaustion or fatigue.

Top 4 out of 1000+ competitors at CalHacks 2015.

MyoMessenger on Github

HappyHour

Pebble Time application for safety that allows users to keep track of how many drinks they’ve had, calculate their BAC in real time, and give them the option to call an Uber to their location at the click of a button.

Won best use of Uber API prize at Hack the North 2015; featured on Devpost‘s top projects.

HappyHour on Github

SafeSteps

Pebble application for safety that allows users to send a call + SMS message to as many friends as they would like simultaneously at a click of a button. The call includes a voice message saying that the user is in trouble, and the SMS message also includes the address of where the user is located.

SafeSteps on Github

ScenicJogger

Web Application that empowers citizens to explore historical landmarks in their cities by intelligently recommending jogging routes that satisfies the user’s desired running distance while routing around historical landmarks.

ScenicJogger on Github
